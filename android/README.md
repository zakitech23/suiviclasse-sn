# SuiviClasse SN – Application Android

Application de suivi des performances des élèves – Sénégal (IEF Bignona 2)

## Télécharger l'APK

1. Cliquer sur l'onglet **Actions** ci-dessus
2. Cliquer sur le dernier workflow **Build APK SuiviClasse SN**
3. Télécharger **SuiviClasse-SN-APK** en bas de la page

## Fonctionnalités
- Page de garde (IA, IEF, École, Maître, Oustas...)
- Saisie des notes : Prétest, Compo 1, 2, 3
- Grille officielle : LC, Maths, Découverte du Monde, Dév. Durable, EPSA, Arabe
- Bulletins par trimestre imprimables
- Synthèse classe avec statistiques
- Proposition de passage annuelle
- Import élèves : Excel, Word, PDF, Image
